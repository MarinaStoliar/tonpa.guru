Новая версия FORM

FORM — это генератор форм для разных бекендов на которой написана страница депозитов pb.ua/depozit. Так как я сейчас обновляю стек для еще одного промышленного клиента, решил выпустить новую версию, которой уже будет 4 года, после появления первых версии промышленного использования.

— github.com/synrc/form — 4.3

Домашняя страница проекта — form.n2o.dev является N2O приложением использующим веб-фреймворк NITRO который обновляет DOM, пересылает ивенты по вебсокету и определяет целевой формат генерации для библиотеки FORM. Потом контрольные элементы NITRO преобразуются в HTML5 средствами самого NITRO, и затем сериализованное как строка передается как управляющая команда для апдейта DOM на N2O JavaScript клиенте, который, доверяя серверу, выполняет только eval.

Приложение использует статические адреса JavaScript клиента N2O рассположенные на Github Pages, а сами формы берутся из исходников рерпозитория библиотеки и посылаются при установлении вебсокет соединения в реакции сообщения init протокола NITRO.
